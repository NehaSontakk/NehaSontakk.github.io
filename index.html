Operationalizing Explainability in Robo Advisory Applications for Regulators.
Sanjana Krishnan, Sahil Deo*, Neha Sontakke*
(* = Equal Contribution)
DGN 2019 Working Paper for the Data Governance Network


Our paper proposes an approach towards developing a ‘RegTech tool’ that can explain the Robo advisors’(RA) decision-making. RAs assign investment options based on a risk profile they generate for users. We use machine learning to reverse engineer the importance, effects and relationships of features used in this black-box algorithm for risk profiling. To provide regulators, who lack the technical knowledge to understand algorithmic decisions, three levels of explanation. These explanations aid regulators to understand how consumer/user data is analysed and to ensure that the risk-profiling done by RA applications comply with the regulations they are subjected to.

Personal information classification using cloud object storage for GDPR security compliance.
Smita Raut, Neha Sontakke
IEEE International Conference on Cloud Computing in Emerging Markets (CCEM) 2018.


We present a solution for personal data classification at the object storage level using cognitive computing. Our work focuses on textual object data and uses Natural Language Processing techniques to distinguish the text containing personal information. Object storage is an evolving form of storage for unstructured data and is gaining popularity. Combined with the substantial progress in the field of cognitive computing, artificial intelligence, and machine learning storage systems can conduct a deep analysis of this unstructured data to identify vulnerabilities.

Hybrid Recommendation Engine with Web Scraping and Sentiment Analysis.
Neha Sontakke, Sabarinath S, Shubham Sangamnerkar, Vishva Iyer
Conference on Machine Learning, Data Science and IOT 2019 organized by IEEE Computer Society


In order to reduce time, effort, and information overload on consumer decision-making while purchasing a product, we propose a robust and scalable hybrid recommendation system - implemented as a combination of Content-Based Recommendation System and Collaborative Filtering Techniques. This system aids the users to purchase smartphones based on features gathered from multiple websites mined using a self-learning web crawler. To further ease the burdern of decision making, the sentiments of users who purchased these items previously are analyzed and displayed.
